Python Foundation

python
numbers = [1, 3, 5, 15, 30, 45]
for i in range(len(numbers)):
    if numbers[i] % 3 == 0 and numbers[i] % 5 == 0:
        numbers[i] = "FizzBuzz"
    elif numbers[i] % 3 == 0:
        numbers[i] = "Fizz"
    elif numbers[i] % 5 == 0:
        numbers[i] = "Buzz"
print(numbers)  # ['1', 'Fizz', '5', 'FizzBuzz', 'FizzBuzz', 'FizzBuzz']

Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. It supports procedural, object-oriented, and functional styles. The reference interpreter (CPython) is open-source and runs on many OSes. Learning Python sets you up for success in data science, web dev, automation scripts, and more.

Syntax Basics

Indentation is code structure.
Blocks are denoted by consistent indentation, not braces or keywords.

Comments: # single-line

"""Multi-line""" (can serve as multiline comments or docstrings)

Variables are dynamically typed.
Just assign and go:

name = "Alice"

Modules are imported with:

import module

from module import item

Functions

Definition & Use

  • Define with: def my_function():
  • Flexible parameters: default values, *args, **kwargs
  • Functions are first-class objects: pass as arguments, return them

Advanced Features

  • Lambda: small anonymous functions lambda x, y: x + y
  • Decorators: functions that modify functions
  • Recursion: use with clear exit condition to avoid stack overflow
